quickstart.py 
from remembra import Remembra

# Initialize client
client = Remembra(api_key="your-key")

# Store a memory with entity extraction
client.store(
    content="John met Sarah at the AI conference in NYC",
    user_id="user-123"
)

# Recall with semantic + entity search
memories = client.recall(
    query="Who did John meet?",
    user_id="user-123"
)
# → Returns: John met Sarah at the AI conference

Up and Running in 5 Minutes

From zero to AI memory in three steps

1

Deploy

One command to run Remembra with Docker:

docker run -d -p 8787:8787 \
  remembra/remembra

Or use docker-compose for production with Qdrant included.

2

Integrate

Choose your integration method:

🐍 pip install remembra
📦 npm install @remembra/client
🤖 claude mcp add remembra
🌐 REST API (any language)
3

Use

Store and recall memories:

# Store
memory.store("User likes dark mode")

# Recall
memory.recall("preferences")
# → "User likes dark mode"

Entities extracted. Graph built. All automatic.
