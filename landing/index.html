quickstart.py 
from remembra import Memory

memory = Memory(user_id="user_123")

# Store memories
memory.store("User prefers dark mode and works at Acme Corp")

# Recall with context
context = memory.recall("What are user's preferences?")
# Returns: "User prefers dark mode. Works at Acme Corp."

Up and Running in 5 Minutes

No complex setup. No configuration headaches.

1

Start Remembra

docker run -d -p 8787:8787 -e OPENAI_API_KEY=sk-... remembra/remembra
2

Install the SDK

pip install remembra
3

Start Remembering

from remembra import Memory

memory = Memory(user_id="user_123")
memory.store("User loves hiking and works at Google")
context = memory.recall("What does user do?")
