Security

How we protect your data and maintain trust.

Our Security Philosophy

Remembra is built with security as a first-class citizen. We believe that AI memory systems must be trustworthy, and that starts with robust security practices.

Data Protection

  • Encryption at Rest: All stored memories are encrypted using AES-256.
  • Encryption in Transit: All API communications use TLS 1.3.
  • User Scoping: Memories are strictly scoped to user IDs, preventing cross-user data leakage.
  • PII Detection: Built-in detection and optional redaction of sensitive data.

Authentication & Access Control

  • API Key Authentication: Secure, scoped API keys with granular permissions.
  • RBAC: Role-based access control with Admin, Editor, and Viewer roles.
  • Rate Limiting: Protection against abuse with configurable rate limits.
  • Audit Logging: Full audit trail of all API operations.

Infrastructure

Self-Hosted Option

Deploy Remembra on your own infrastructure with full control over your data. No data ever leaves your network.

Cloud Hosting

Our managed cloud service runs on secure, SOC 2 compliant infrastructure with regular security audits.

Compliance & Privacy

  • GDPR Ready: Full support for data export and deletion (right to be forgotten).
  • Data Portability: Export all your memories in JSON, CSV, or JSONL format.
  • No Training on Your Data: Your memories are never used to train AI models.
