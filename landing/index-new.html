Add memory in minutes, not months

Remembra's SDK handles the complexity. You focus on building. Entity extraction, embedding, and graph updates happen automatically.

View Quickstart → 
from remembra import Memory

memory = Memory()

# Store a memory
memory.store("User prefers dark mode")

# Recall relevant context
results = memory.recall("What are the user's preferences?")

# Entities extracted automatically
print(results.entities)  # ['User']
