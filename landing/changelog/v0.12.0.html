Remembra v0.12.0

User Profiles, Smart Auto-Forgetting & Browser Extension

✨ Added

👤 User Profiles API

Aggregated user intelligence at your fingertips.

  • GET /api/v1/users/{user_id}/profile
  • Returns aggregated facts, activity metrics, and top topics
  • Memory count, last active date, entity breakdown
  • Perfect for personalization and user insights

⚡ Smart Auto-Forgetting

35+ temporal patterns automatically set TTL for you.

  • "meeting tomorrow" → 36h TTL
  • "call next week" → 8 days TTL
  • "deadline in 2 hours" → 3h TTL
  • Works with relative dates, specific times, and duration phrases
  • Zero config — just store memories naturally

🚫 Strict Mode 410 GONE

Opt-in mode for explicit expiry awareness.

  • Enable via strict_mode: true in config
  • Expired memory requests return 410 GONE instead of silent accept
  • Agents can handle expiration explicitly
  • Prevents stale data from silently being used

⏰ Event-Driven Expiry

Explicit timestamps for precise memory lifecycle.

  • New expires_at parameter on store endpoint
  • ISO 8601 timestamp: "2026-03-25T14:00:00Z"
  • Takes precedence over TTL when both specified
  • Perfect for event-driven workflows

🔄 Shadow TTLs Client-Side

SDK caches expiry times locally to reduce server calls.

  • SDK maintains local expiry cache
  • Skip recall for known-expired memories
  • Reduces API calls by up to 40%
  • Automatic cache invalidation on store

🌐 Browser Extension

Chrome Manifest V3 extension for AI chat interfaces.

  • Works with ChatGPT, Claude, and Perplexity
  • Auto-injects memory context into conversations
  • One-click memory storage from any chat
  • Secure local API key storage
  • Available on Chrome Web Store

📦 Installation

# Via pip
pip install remembra==0.12.0

# Via Docker
docker pull remembra/remembra:0.12.0

# Quick start
remembra-install --all

# Browser Extension
# Install from Chrome Web Store: "Remembra for AI"

📖 New API Examples

User Profile

curl -H "Authorization: Bearer rem_xxx" \
  https://api.remembra.dev/api/v1/users/user_123/profile

Store with expires_at

curl -X POST https://api.remembra.dev/api/v1/memories \
  -H "Authorization: Bearer rem_xxx" \
  -d '{"content": "Meeting at 2pm", "expires_at": "2026-03-22T15:00:00Z"}'
