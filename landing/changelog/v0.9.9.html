Remembra v0.9.9

Temporal Knowledge Graph & 11 MCP Tools

✨ Added

⏳ Temporal Knowledge Graph

Bi-temporal relationship model for tracking relationship history.

  • Relationships track valid_from, valid_to, and superseded_by
  • Point-in-time queries: "Where did Alice work in January 2022?"
  • Contradiction detection: new relationships auto-supersede old ones
  • Full relationship history preserved

🛠️ 6 New MCP Tools (11 Total)

Expanded MCP server from 5 tools to 11 tools.

  • update_memory — Update content without delete+recreate
  • search_entities — Search the entity graph by name, type, or alias
  • list_memories — Browse stored memories without search query
  • share_memory — Cross-agent memory sharing via Spaces
  • timeline — Temporal browsing filtered by entity and date range
  • relationships_at — Query entity relationships at a specific point in time

📊 Entity Graph Visualization

Interactive force-directed graph with react-force-graph.

  • Flowing particle effects on relationship edges
  • Entity nodes colored by type
  • Click-to-explore entity neighborhoods

🔄 Contradiction Detection

Automatic relationship supersession with full history preservation.

  • Exclusive relationship types (WORKS_AT, SPOUSE_OF) detect contradictions
  • "Alice works at Meta" automatically supersedes "Alice works at Google"
  • find_contradicting_relationships() for conflict queries
  • get_relationship_history() for full timeline

📦 SDK Client Methods

Python SDK expanded with new methods.

  • memory.update(memory_id, content, metadata) — Calls PATCH endpoint
  • memory.list_entities(entity_type, limit) — Calls entity list endpoint

🔄 Changed

• MCP server instructions updated to reflect 11 available tools

• Entity graph retrieval now supports temporal filtering

• Extraction prompt now extracts WHEN relationships started/ended
