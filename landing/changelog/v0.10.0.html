Remembra v0.10.0

Universal Agent Installer & Setup Diagnostics

✨ Added

🤖 Universal Agent Installer

One command configures ALL your AI tools.

  • remembra-install --all — Auto-detect and configure all agents
  • remembra-install --agent claude-desktop — Configure specific agent
  • remembra-install --detect — List installed agents
  • Safe config merging — preserves existing MCP configurations
  • Centralized credentials in ~/.remembra/credentials

🔍 Setup Diagnostics

Troubleshoot connection issues with clear failure labels.

  • remembra-doctor all — Scan all detected agents
  • remembra-doctor codex — Diagnose specific agent
  • Clear failure labels: dns_failure, sandbox_blocked, auth_failure, timeout
  • Health probe + recall test for end-to-end verification

🌉 Local Bridge

HTTP proxy for sandboxed agents that can't reach external URLs.

  • remembra-bridge — Start local proxy on 127.0.0.1:8765
  • Forwards requests to remote Remembra API
  • API key injection for authentication
  • Auto-configured by installer for Codex CLI

🛡️ Security Hardening

Enterprise-grade security improvements.

  • RBAC permissions enforced on all memory endpoints
  • Generic exception handler — no stack traces leaked
  • API key caching for reduced latency
  • Webhook SSRF protection
  • 2FA/MFA settings UI in dashboard

🔧 Supported Agents

✓ Claude Desktop
✓ Claude Code
✓ Codex CLI
✓ Cursor
✓ Windsurf
✓ Gemini

🐛 Fixed

  • Rate limit removed from /health endpoint (was blocking monitoring)
  • RBAC now uses inline permission checks instead of Depends()

📦 Installation

Upgrade pip

pip install --upgrade remembra

Configure All Agents

remembra-install --all --api-key rem_xxx

Upgrade Docker

docker pull remembra/remembra:0.10.0
