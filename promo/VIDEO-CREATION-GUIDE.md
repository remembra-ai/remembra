# Remembra Promo Video — Quick Creation Guide

**Goal:** 60-second professional promo video  
**Assets Ready:** ✅ Voiceover, ✅ Script, ✅ Storyboard

---

## 🎬 OPTION 1: Magic Hour (Fastest — 15 min)

1. Go to [magichour.ai](https://magichour.ai)
2. Sign up (free tier available)
3. Use **Text-to-Video** tool
4. Paste this prompt:

```
Professional tech product video. Dark mode aesthetic. 
Cinematic shots of code on screens, developers working, 
abstract AI neural network visualizations, data flowing.
Modern startup energy. Clean typography.
Style: Tech product launch video like Vercel or Linear.
```

5. Generate 4-5 clips (5 seconds each)
6. Download clips
7. Upload to CapCut/DaVinci with our voiceover

**Voiceover location:** `/Users/dolphy/projects/remembra/promo/assets/voiceover-60s.mp3`

---

## 🎬 OPTION 2: HeyGen (AI Avatar — 20 min)

1. Go to [heygen.com](https://heygen.com)
2. Use free trial
3. Choose a professional avatar (male, tech-savvy look)
4. Paste script from `/promo/video-script-60s.md`
5. Add screen recordings as overlays:
   - Terminal showing `docker run remembra`
   - Dashboard demo
   - Code examples
6. Export and done

---

## 🎬 OPTION 3: Canva (Free — 30 min)

1. Go to [canva.com/videos](https://canva.com/videos)
2. Search template: "Tech Product Launch"
3. Customize with:
   - Remembra logo
   - Feature bullet points
   - Code screenshots
4. Add voiceover MP3
5. Export
